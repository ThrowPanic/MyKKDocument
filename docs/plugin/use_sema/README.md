## 插件用法

此插件为ManlyuMarco创作。用作于工作室上，在工作室右下角显示一个与调整模型的同步的控制箭头如下图(\*1)

![](./media/222.png)（\*1)

方便在模型过多时或被掩盖控制箭头无法点击到时可以直接从这边调整

注意，方向xyz的三个方向有的时候会和实际对不上，我一般用于旋转和调整大小，移动的话有的时候不好用
